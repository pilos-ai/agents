Privacy Policy

Last updated: February 22, 2026

Overview

Pilos Agents is a local-first desktop application. Your conversations, project files, and AI interactions stay on your machine. We do not operate cloud servers that process or store your development data.

This policy covers what limited data we do collect through the website, license server, and optional services.

Data We Collect

Desktop Application

The Pilos Agents desktop app:

  • Does not send your conversations, code, or project data to our servers
  • Does not include telemetry, analytics, or crash reporting by default
  • Does communicate with Anthropic's API using your API key — refer to Anthropic's privacy policy for how they handle API requests
  • Does validate license keys against our license server (sends only the key string, no other data)

License Server

When you purchase a Pro or Teams plan, we store:

  • Email address (provided during checkout)
  • License key
  • Plan type and seat count
  • Subscription status and expiry date

This data is used solely for license validation and billing. We do not sell or share it with third parties.

Website

The pilosagents.com website may use privacy-respecting analytics (e.g., Plausible or Vercel Analytics) to understand page views and referral sources. No cookies are used for tracking. No personal data is collected from website visitors.

Third-Party Services

  • Stripe — Processes payments for Pro and Teams plans. Stripe collects payment information directly; we never see your full card number. See Stripe's privacy policy.
  • Anthropic — Processes AI requests via the Claude API using your API key. See Anthropic's privacy policy.
  • GitHub — Hosts the open source repository. Interaction with GitHub is governed by GitHub's privacy policy.
  • Vercel — Hosts this website. See Vercel's privacy policy.

MCP Server Connections

When you configure MCP integrations (GitHub, Slack, Jira, etc.), the desktop app connects directly to those services using credentials you provide. We do not proxy, log, or store any data from these connections.

Data Retention

  • App data: Stored locally on your machine. Delete it anytime by removing the app data folder.
  • License data: Retained for the duration of your subscription plus 30 days after cancellation.
  • Website analytics: Aggregated, non-personal data retained for up to 24 months.

Your Rights

You may request to:

  • Access any personal data we hold about you
  • Delete your license server account and associated data
  • Export your data in a portable format

Contact privacy@pilosagents.com for any data requests.

Changes to This Policy

We may update this policy from time to time. Significant changes will be announced on our GitHub repository and website. Continued use of Pilos Agents after changes constitutes acceptance.

Contact
