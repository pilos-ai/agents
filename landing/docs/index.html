Documentation

Everything you need to get started with Pilos Agents and make the most of your AI development team.

Installation #

Pilos Agents is a native desktop app available for macOS, Windows, and Linux. Download the latest release from GitHub:

  1. Go to github.com/nicepkg/pilos-agents/releases
  2. Download the installer for your platform (.dmg for macOS, .exe for Windows, .AppImage for Linux)
  3. Run the installer and open Pilos Agents

Prerequisite: You need the Claude Code CLI installed. Run npm install -g @anthropic-ai/claude-code to install it.

API Key Setup #

Pilos Agents uses your own Anthropic API key. We never see, store, or charge for your AI usage.

  1. Get an API key from console.anthropic.com
  2. Open Pilos Agents and go to Settings → API Key
  3. Paste your key and click Save

The key is stored locally on your machine and is never transmitted to our servers.

First Project #

Once your API key is set up, create your first project:

  1. Click + New Project in the sidebar
  2. Select a folder on your machine — Pilos Agents will detect your codebase automatically
  3. Start a new conversation and ask your AI team anything about the project

Your conversation history, project context, and agent memory are all stored locally in SQLite. Switch between projects anytime without losing context.

Multi-Agent Mode #

Multi-Agent Mode lets you assign specialized AI agents to every conversation. Each agent has a unique role, personality, and area of expertise.

How it works

  • Choose from 22 pre-built personas: CEO, CTO, Product Manager, Developer, Designer, QA, DevOps, Security, and more
  • Each agent has a distinct system prompt that shapes its responses
  • Messages are color-coded and attributed by agent
  • Agents can disagree, build on each other's ideas, and provide diverse perspectives

Solo vs Team mode

Toggle between Solo mode (single agent, included in Free) and Team mode (multiple agents, Pro feature) per project. Solo mode is great for focused tasks; Team mode excels at planning, code review, and complex problem-solving.

Terminal #

The integrated terminal is powered by xterm.js and runs directly inside Pilos Agents. No need to switch windows.

  • Full terminal emulation with color support
  • Multiple terminal tabs with per-tab working directory
  • Claude can execute commands and display output inline
  • Working directory is synced with your project folder

MCP Integrations #

Pilos Agents supports the Model Context Protocol (MCP) for connecting Claude to external tools and services.

Built-in templates

14+ one-click setup templates are included:

  • GitHub — Issues, PRs, code search
  • Slack — Send messages, search channels
  • Databases — PostgreSQL, SQLite, Supabase
  • Kubernetes — Clusters, deployments, Helm
  • Terraform — Plan, validate infrastructure
  • Security — SAST, dependency audit, secrets scanning
  • Figma — Inspect designs and tokens
  • Monitoring — Grafana metrics, logs, traces
  • And more: Linear, Analytics, i18n, Custom servers

Custom MCP servers

You can connect any MCP-compliant server via stdio, HTTP, or SSE transport. Add custom servers in Settings → MCP Servers.

Jira Integration #

Connect your Jira instance with OAuth for full project management integration:

  • Sync epics, sprints, and stories
  • Create and manage stories with acceptance criteria
  • Track story coverage and push updates to Jira
  • View sprint boards directly in the app

Set up in Settings → Integrations → Jira.

Project Memory #

Every conversation is stored locally in SQLite. Pilos Agents maintains persistent project memory so your AI team never forgets context.

  • Full conversation history with search
  • Per-project agent configurations
  • Automatically loads relevant context when you return to a project
  • All data stays on your machine — no cloud sync required

Permission Modes #

Control what Claude can do in your project with granular permission modes:

  • Ask — Claude asks before every action (safest)
  • Auto-approve reads — Claude can read files freely, asks before writing
  • Auto-approve all — Claude acts autonomously (fastest)

Permission modes are set per-project in project settings.

License Activation #

After purchasing a Pro or Teams plan, you'll receive a license key via email (format: PILOS-PRO-XXXX-XXXX).

  1. Open Pilos Agents and go to Settings → License
  2. Paste your license key
  3. Click Activate — Pro features unlock instantly

Your license is validated against our license server and cached locally. The app works offline after initial activation.
